# Zero Trust Architecture Quiz

ゼロトラストアーキテクチャの理解度を試すクイズアプリケーション（Next.js）

## 概要

このアプリケーションは、ゼロトラストアーキテクチャの主要コンポーネントとそれらの間のやり取りについて理解を深めるためのクイズアプリです。

### ゼロトラストアーキテクチャにおける位置づけ

このNext.jsアプリケーションは、ゼロトラストアーキテクチャにおいて**Resource（リソース）**として位置づけられます。

- **Resource**: ユーザーがアクセスしようとしている保護対象のサービスやデータ
- 実際のゼロトラスト環境では、Envoy（PEP）の背後で公開され、OPA（PDP）と連携してアクセス制御される
- ユーザー（Subject）がこのリソースにアクセスする際、PEP/PDPによる認証・認可が行われる

## 出題内容

以下のゼロトラストアーキテクチャの主要コンポーネントに関する問題が出題されます：

- **Policy Enforcement Point (PEP)**: データプレーンに位置し、アクセスを強制するコンポーネント
- **Policy Decision Point (PDP)**: 制御プレーンに位置し、アクセスを許可/拒否する決定を下すコンポーネント
- **Policy Information Point (PIP)**: PDPがポリシー決定に必要な情報を供給するサポートコンポーネント
- **Resource（リソース）**: 保護対象のサービスやデータ
- **Subject（サブジェクト）**: リソースにアクセスしようとするユーザーとエンドポイント
- **メッセージフロー**: I(N)（初期接続）、S(X)（セッション管理）、R(X)（リソース管理）

## セットアップ

### 必要な環境

- Node.js 18以上
- npm, yarn, pnpm, または bun

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリケーションにアクセスします。

## 使い方

1. トップページで「クイズを開始する」ボタンをクリック
2. 各問題に回答を選択
3. 「回答を確定する」ボタンをクリック
4. 解説を確認して「次の問題へ」をクリック
5. 全問終了後、結果画面でスコアと問題別の結果を確認
